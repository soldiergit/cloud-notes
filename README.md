# CloudNotes 作者：黄结

## 2020年春季大三第二学期，Android期末课设：这是一个基于 Android 和 MySql 的云笔记本程序。

### 实现功能:   
  1. **用户的注册登录退出模块**
  2. **用户信息的查找更改**操作
  3. **笔记列表的查看**操作
  4. **笔记列表的删改**操作
